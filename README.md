### Prerequisites

```bash
node --version    # 18+ required
npm --version     # 9+ recommended
```

### Installation

```bash
# ✦ Clone the repository
git clone https://github.com/ravi-c029/portfolio.git
cd portfolio

# ✦ Install dependencies
npm install

# ✦ Start development server
npm run dev
# → http://localhost:3000

# ✦ Build for production
npm run build

# ✦ Preview production build
npm run start
```

### Environment Variables *(if applicable)*

```env
NEXT_PUBLIC_SITE_URL=https://www.ravimedical.in
NEXT_PUBLIC_EMAIL=ravi.keshari029@gmail.com
```

> 💡 No heavy environment setup needed — this is a static portfolio!
